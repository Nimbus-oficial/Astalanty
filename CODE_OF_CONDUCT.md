# Code of Conduct

## Our Standard

Astalanty aims to maintain a professional, respectful and technically focused collaboration environment.

Expected behavior:

- communicate respectfully;
- assume good intent;
- focus criticism on technical work, not individuals;
- avoid harassment, threats, discrimination or abuse;
- respect privacy and security boundaries.

## Unacceptable Behavior

Unacceptable behavior includes:

- personal attacks;
- harassment or intimidation;
- publishing private information;
- attempting to extract secrets or credentials;
- knowingly misleading contributors or users about the state of the project.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions or block participants who violate this code of conduct.
