# Astalanty TypeScript SDK

Official TypeScript SDK for the Astalanty technical MVP economic core.

The SDK hides direct Solidity contract interaction behind a small API for dapps:

```ts
const sdk = new AstalantySDK(config);

await sdk.connect(window.ethereum);
await sdk.createSmartAccount();
await sdk.approvePaymaster();
await sdk.executeSponsoredTransaction();
```

## Install

Inside the monorepo:

```powershell
pnpm --filter @astalanty/sdk build
```

## Configuration

Use the deployment JSON generated by `packages/contracts/scripts/deploy-mvp.cjs`:

```ts
import { AstalantySDK, configFromDeployment } from "@astalanty/sdk";
import deployment from "../deployments/arbitrumSepolia-421614-latest.json" assert { type: "json" };

const sdk = new AstalantySDK(
  configFromDeployment(deployment, "https://sepolia-rollup.arbitrum.io/rpc")
);
```

Or configure manually:

```ts
const sdk = new AstalantySDK({
  network: {
    name: "arbitrumSepolia",
    chainId: 421614n,
    rpcUrl: "https://sepolia-rollup.arbitrum.io/rpc"
  },
  contracts: {
    mockUsdc: "0x...",
    ausdToken: "0x...",
    feeManager: "0x...",
    paymaster: "0x...",
    smartAccountFactory: "0x..."
  }
});
```

## Public API

### `connect(wallet)`

Connects the SDK to an ethers `Signer` or browser wallet provider.

```ts
await sdk.connect(window.ethereum);
```

### `createSmartAccount(options?)`

Creates a Smart Account for the connected owner, or returns the existing one.

```ts
const account = await sdk.createSmartAccount();
```

### `getSmartAccount(owner?)`

Reads the Smart Account registered for an owner.

```ts
const account = await sdk.getSmartAccount();
```

### `getMockUSDCBalance(owner?)`

Reads Mock USDC balance.

```ts
const balance = await sdk.getMockUSDCBalance();
```

### `getAUSDBalance(owner?)`

Reads AUSD balance.

```ts
const balance = await sdk.getAUSDBalance("0x...");
```

### `approvePaymaster(options?)`

Approves the Paymaster to spend Mock USDC from the Smart Account by calling `SmartAccount.execute`.

```ts
await sdk.approvePaymaster();
```

### `quoteSponsoredTransaction(options?)`

Reads the Fee Manager quote for the MVP sponsored operation.

```ts
const quote = await sdk.quoteSponsoredTransaction();
```

### `executeSponsoredTransaction(options?)`

Executes `Paymaster.sponsorDemoOperation` using the active Smart Account as account and payer.

```ts
const result = await sdk.executeSponsoredTransaction();
```

### `getEvents(options)`

Queries principal MVP events.

```ts
const events = await sdk.getEvents({
  event: "PaymasterFeeSettled",
  fromBlock: 0,
  toBlock: "latest"
});
```

## Complete MVP Flow

```ts
await sdk.connect(window.ethereum);

const { smartAccount } = await sdk.createSmartAccount();
const before = await sdk.getMockUSDCBalance(smartAccount);

await sdk.approvePaymaster();

const result = await sdk.executeSponsoredTransaction();
const ausd = await sdk.getAUSDBalance();

console.log({
  smartAccount,
  mockUsdcBefore: before.formatted,
  operationId: result.operationId,
  ausdBalance: ausd.formatted
});
```

## Front-End Integration Checklist

1. Load deployment JSON for the selected network.
2. Instantiate `AstalantySDK` with `configFromDeployment`.
3. Connect MetaMask with `sdk.connect(window.ethereum)`.
4. Show owner address and Smart Account address.
5. Call `sdk.createSmartAccount()` if none exists.
6. Show Mock USDC and AUSD balances.
7. Call `sdk.approvePaymaster()` before the demo operation.
8. Call `sdk.executeSponsoredTransaction()`.
9. Show `operationId`, transaction hash, quote, and settlement events.
10. Link transaction hashes to Arbiscan Sepolia after public deploy.

## MVP Limitations

- `executeSponsoredTransaction` wraps the current `sponsorDemoOperation`, not a full ERC-4337 EntryPoint flow.
- Mock USDC must already exist in the Smart Account before approval and sponsorship.
- Contract verification and explorer links are outside this SDK.
- The SDK is intentionally focused on the MVP economic core only.
